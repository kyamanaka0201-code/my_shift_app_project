## ⚙️ 技術スタック
- **バックエンド**: Python, Django 5.2
- **データベース**: PostgreSQL
- **フロントエンド**: HTML, CSS, JavaScript
- **データ可視化**: Matplotlib, japanize-matplotlib
- **デプロイ**: Render

## ✨ 主な機能
- シフトの自動作成（勤務日数・連続勤務制約を考慮）
- 月別シフト表の表示
- 給与グラフの可視化
- Excel/CSV形式でのエクスポート
- 管理者画面での従業員管理

## 📝 開発の背景
以前シフト制の職場で働いていた際、シフト作成の担当者が毎回大変そうに作成していた様子を見て、もっと便利にできるツールがあればいいなと思い開発しました。

##  ❕ 改善点・今後の展望
- 柔軟なシフト対応の実現
 「この日は○時までしか勤務できない」など、個々の勤務時間に合わせた柔軟なシフトを組めるよう、早番・中番・遅番以外の細かな時間設定機能を追加していきます。
- 人員バランスの最適化
 同じ人ばかりが入るなどの偏りを自動的に調整し、より公平でバランスの取れたシフトを生成できるようにアルゴリズムを改善していきます。
- 現実的なシフト設計の強化
  出勤日だけでなく、休日のバランスも考慮し、連休が続きすぎないなど、実際の勤務環境に近いシフトが作成できるツールを目指します。

- ✨ 目指すのは、「誰でも簡単に・効率よく・現場に合ったシフトを作れるツール」です。
